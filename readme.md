GOSS'IP TEA est un forum développé en GO permettant aux utilisateurs de publier des posts, commenter, liker et interagir avec les autres membres.

Fonctionnalités :

- Comptes utilisateurs:
    - Inscription
    - Connexion / Déconnexion
    - Modification du profil
    - Suppression du compte 
    - Avatar personnalisé
    - Réinitialisation du mot de passe par mail

- Posts :
    - Création de posts
    - Modification de posts
    - Affichage des posts
    - Détail d'un post
    - Like / Unlike
    - Commentaires
    - Suppression de commentaires

- Recherche :
    - Recherche d'utilisateurs 
    - Recherche de posts
    - Tri :
        - Par date, par like, par commentaire
        - Ordre croissant / décroissant



- Technologies utilisées :
    - GO
    - HTML / CSS
    - SQLite (SQL)
    - bcrypt (hachage des mots de passe)
    - JavaScript (fetch api)

Pour lancer le projet en local:
git clone https://github.com/Lyon-Ynov-Campus/forum-gossip.git
cd forum-gossip 
go run .
Puis il faut aller sur : http://localhost:8080
